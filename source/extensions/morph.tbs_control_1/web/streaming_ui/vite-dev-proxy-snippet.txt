[Vite 개발 서버(5173)에서 /api 를 Kit 브리지(8720)로 넘기기]

vite.config.ts 의 defineConfig({ ... server: { ... } }) 안에 예:

  server: {
    proxy: {
      "/api": {
        target: "http://127.0.0.1:8720",
        changeOrigin: true,
      },
    },
  },

이렇게 하면 React 쪽에서 fetch("/api/state") 가 5173 으로 가고, Vite 가 8720 으로 전달합니다.
이 경우 .env 에 다음을 두면 됩니다 (빈 값 = 상대 경로):

  VITE_TBS_KIT_API_BASE=

프록시를 쓰지 않고 브라우저가 직접 Kit 에 붙이려면:

  VITE_TBS_KIT_API_BASE=http://127.0.0.1:8720

(브라우저 CORS 는 kit_remote_http_bridge 가 /api 에 Access-Control-Allow-Origin: * 를 붙입니다.)
